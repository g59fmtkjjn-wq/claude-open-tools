# Claude Open Tools

Open-source tools, utilities, prompts and resources for **Claude AI** and **Claude Code**.

## About

This project aims to make Claude more accessible and powerful for developers by providing:
- Useful prompts and templates
- CLI tools and utilities
- Best practices and guides
- Agentic workflow examples

## Features
- Curated prompt library
- Ready-to-use Claude Code examples
- Community resources (French + English)
